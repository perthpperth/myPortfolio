Project

  • Coffee Machine Project

              The CoffeeMachine application presents a user-friendly graphical interface for a cafe ordering system. Users input their names and select coffee options through checkboxes, dynamically updating the total price display. The interface responds to various user interactions, generating an order summary upon pressing enter. Notably, a countdown timer triggers a summary display after 60 seconds, showcasing the application's responsiveness. Overall, it offers a simple and functional illustration of a cafe ordering system through Java GUI.
    Role: UI design, Frontend

  • Bookbelay: Book Reservation System

              Bookbelay is a project aimed at modernizing library management. It handles intricate book data and user records. Its functionalities span user registration, book management, reservation, and checkout processes, supported by advanced search, genre categorization, and secure authentication. The system is designed for user satisfaction, offers intuitive interfaces, QR code integration, and scalability, representing an important shift towards user-centered, user-efficient, and accessible library services.
    Role: Database Design, UI Design, Frontend

  • The SA Website Project

              This project seeks to modernize and enhance the online presence of the SIIT SA through a comprehensive and contemporary website redesign. The current website is outdated and in need of a user-friendly and visually appealing interface to better serve its audience. Additionally, a streamlined and efficient content management system will be established to facilitate seamless updates and maintenance, ensuring that the website remains current and relevant.
    Role: UI Design

  • The SIIT’s Activities Application Project (On-going)

              The application serves as a centralized hub for faculty activities, streamlining the user experience for both students and staff. It includes a robust registration system that simplifies the application process for users. Moreover, the application facilitates communication by enabling staff members to make announcements efficiently. This multifaceted solution aims to enhance accessibility, organization, and communication within the faculty, fostering a more seamless and informed experience for all users.
